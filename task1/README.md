# Vistula University - Spring Boot Task 1

## Project Overview
This is a basic Spring Boot application developed as part of the Java Spring Framework course at Vistula University. The project demonstrates the setup of a web controller, request mapping, and the integration of Thymeleaf templates for dynamic web content.

## Features
* **Hello Controller:** A basic `@RestController` implementation returning simple text.
* **Greeting Page:** A `@Controller` using Thymeleaf to render an HTML view.
* **Dynamic Content:** Uses `@RequestParam` to personalize the greeting via the URL.
* **Static Resources:** Integration of static images (Vistula University logo).

## Technologies Used
* **Java 26**
* **Spring Boot 4.0.6**
* **Maven** (Dependency Management)
* **Thymeleaf** (Template Engine)
* **IntelliJ IDEA** (Development Environment)

## How to Run
1.  Ensure you have **JDK 26** installed.
2.  Open the project in IntelliJ IDEA.
3.  Allow Maven to download the necessary dependencies.
4.  Run the `Task1Application.java` file.
5.  Access the application in your browser at:
    * `http://localhost:8080/greeting`
    * `http://localhost:8080/greeting?name=YourName`
